# Blackbird

Threat Intelligence Enrichment Platform

---

## Overview

Blackbird is an open-source threat intelligence platform focused on indicator enrichment, intelligence correlation, and investigation support.

The platform aggregates intelligence from multiple providers, normalizes heterogeneous threat data into a common model, and produces transparent risk assessments with complete source attribution.

Blackbird is designed around a simple principle:

> Intelligence collection, enrichment, and evaluation should be explainable.

Every score, finding, and assessment generated by the platform should be traceable to the underlying evidence.
